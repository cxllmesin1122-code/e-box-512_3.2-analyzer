✅ MALWARE DETECTOR — 5 THREAT CATEGORIES

   🦠 MALWARE
      • Generic malicious code detection
      • Signature + mathematical model analysis
      • Per-category confidence scoring

   🔧 ROOTKIT
      • Kernel-mode threat detection
      • System-level hooking patterns
      • Operational probability calculation

   🎯 AIMBOT
      • Game cheat detection
      • Timing loop analysis
      • Vector math pattern recognition

   🛡️ ANTICHEAT
      • Anti-cheat bypass detection
      • Integrity check patterns
      • Kernel-mode AC pattern matching

   💉 VIRUS
      • Packed/self-replicating detection
      • UPX/RAR/ZIP payload patterns
      • High-entropy section analysis


✅ PER-CATEGORY METRICS (ALWAYS DISPLAYED)

   📊 Confidence Score
      • Range: 0.0 - 1.0
      • Per-detection value shown in GUI

   📊 Operational Probability
      • Execution likelihood (0-100%)
      • Formula: score × (0.6 + 0.4 × markov_norm)
      • Shown for each detection

   📊 Overall File Confidence
      • Per-category aggregate (malware/rootkit/aimbot/anticheat/virus)
      • Combines window-level + structural signals
      • Displayed as percentage breakdown


✅ BINARY ANALYSIS PIPELINE

   1. EBox512 Entropy Scanner (5-Gate System)
      • Shannon entropy analysis
      • Chi-squared distribution
      • KL-divergence uniformity
      • Autocorrelation detection
      • Spectral dominance

   2. Malware Detector
      • Signature distance matching
      • Kolmogorov-Smirnov test
      • Fourier periodicity analysis
      • Markov chain entropy
      • File-level structural heuristics

   3. Binary Parser
      • Format detection (ELF, PE, Mach-O, DEX, ZIP, etc.)
      • Section entropy analysis
      • Import/symbol extraction
      • Embedded region scanning

   4. Decompiler
      • Capstone disassembly (x86/ARM/MIPS)
      • Function boundary detection
      • Pseudo-C code generation
      • Cross-reference mapping

Issue: GUI won't start
   → Verify: python -c "import gui_app; print('OK')"
   → Check: All dependencies installed (pip install -r requirements.txt)

Issue: Malware detector shows 0 detections
   → This is normal for clean/legitimate binaries
   → Try test_binary_malware.elf to see detections

Issue: GPU not detected
   → This is OK - NumPy CPU fallback is automatic
   → To enable GPU: pip install cupy-cuda11x (or your CUDA version)

Issue: Tests fail
   → Run: python test_suite.py
   → Check output for specific failures
